### 🚀现在就要使用？

1. 安装Nodejs
2. Clone本项目
3. 编辑`/src/config/xxx.yml`文件
4. 安装依赖

```bash
npm install
npm run build
```

5. 生成的静态网页就会出现在`/build`文件夹中
